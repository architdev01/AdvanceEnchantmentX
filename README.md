## 🧠 Advanced Systems

### 🧬 XP & Progression

Every enchant gains XP through usage.
As you fight or mine, enchants automatically **level up**, increasing their power.

---

### 🧪 Fusion System

Combine enchantments to unlock powerful new ones.

Example:
FireAura + LightningStrike → ⚡🔥 StormFlame

---

### 🌍 Environment Sync

Enchants adapt to the world:

* ❄️ FreezeAura stronger in water
* 🔥 FireAura stronger in Nether

---

### ⚡ Active Abilities

Not just passive effects — interact with your enchants:

* Right-click → weapon abilities
* Shift → armor abilities

## 🎮 GUI System

---

## ⚙️ Commands

### Player

```
/aex menu
/aex stats
```

### Admin

```
/aex give <player> <enchant> <level>
/aex apply <player> <enchant> <level>
/aex reload
```
